Endpoints

GET /posts
List all posts
GET /posts/latest
List latest 3 posts (sorted by creation date)
GET /posts/{id}
Show a specific post
GET /posts/of/{author}
List all posts from an author
GET /posts/in/{category}
List all posts belongs to a category
POST /posts
Create a post
PUT /posts/{id}`
Modify a post

POST /posts and PUT /posts accept JSON data with following fields: "author", "title", "text", "categories".

"categories" field is a list of strings while all other fields arestrings.
